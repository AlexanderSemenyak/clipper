Polygons

Del.» TPolygons = array of TPolygon;

C++ » typedef std::vector< Polygon > Polygons;

C#  » using Polygons = List<List< IntPoint >>;


This structure represents a list of polygons. Some of the contained polygons may be holes, but this is determined by orientation, their locations relative to other polygons and the specified polygon filling rule.
